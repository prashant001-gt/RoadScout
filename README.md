# RoadScout
This project focuses on automatic road segmentation from satellite images using deep learning. It extracts roads from other elements like vegetation and land, generating precise road masks for mapping and accessibility analysis. Useful for disaster response, urban planning, and navigation.

This project implements a UNET with a backbone of VGG19 and VG16 to perform image segmentation on images of roads, this project used the Road segmentation dataset from kaggle.
## Description

```
1.This project employs a UNET model with VGG16 , VGG19 as encoder backbones and inbuilt pytorch vg16 based model for comparison.
2.Extensive data analysis and augmentation techniques were applied to improve model robustness and generalization.
3.Compared to the traditional UNET architecture, both VGG16 and VGG19 enhance segmentation performance.
4.The project has applications in road surveillance, navigation, and intelligent transportation systems.
5.It also contributes to autonomous driving by improving road segmentation accuracy.
```
## Dataset
```
 1. The dataset was obtained from kaggle.
 2. Link : https://www.kaggle.com/datasets/trainingdatapro/roads-segmentation-dataset/data 
 3. This dataset comprises a collection of images captured through DVRs (Digital Video Recorders) showcasing roads. 
     Each image is accompanied by segmentation masks demarcating different entities (road surface, cars, road signs, marking and background) within the scene.
```
